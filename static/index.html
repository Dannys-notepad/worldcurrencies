World Currencies API

An API free for developers

World Currencies is an API to lookup all Currencies around the world, its a free API built with the intent of helping developers with all World Currencies they could intergrate in their projects

A simple GET request can be used to fetch data from the API, via CURL, a Broswer or Postman

Getting Currencies

Getting all

URL: https://worldcurrencies-api.onrender.com/api/currencies

Returns a json response with all available currencies, including their abbreviations, symbols and fullname

Requesting With a limit

URL: https://worldcurrencies-api.onrender.com/api/currencies/limit/:limit

Example: https://worldcurrencies-api.onrender.com/api/currencies/limit/15

Returns numbers of currncies specified in the limit

Searching Currencies

Currencies can be searched for in two ways

  1. Through their Abbreviation
  2. Through their Name

Searching With Abbreviation

URL: https://worldcurrencies-api.onrender.com/api/currencies/search/:abb

Example: https://worldcurrencies-api.onrender.com/api/currencies/search/ngn

Returns the specified currency and its properties

Returns a 404 response if the searched currency was not found

Searching With Currency Name

URL: https://worldcurrencies-api.onrender.com/api/currencies/search/:name

Example: https://worldcurrencies-api.onrender.com/api/currencies/search/nigeriannaira

Returns the specified currency and its properties

Returns a 404 response if the searched currency was not found
